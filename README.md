# Adaptive Morse Code Intelligence

## Problem Statement
In emergency situations, communication becomes difficult due to lack of network or connectivity.

## Solution
This project uses AI-based Morse code detection and conversion to enable communication during emergency situations.

## Technologies Used
- Python
- NumPy
- TensorFlow
- PyAudio

## How to Run
1. Install required libraries
2. Run main.py
3. Speak or input Morse signal
